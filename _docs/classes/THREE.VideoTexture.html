Class VideoTexture

Creates a texture for use with a video.

Remarks

Note: After the initial use of a texture, the video cannot be changed Instead, call .dispose() on the texture and instantiate a new one.

Example

// assuming you have created a HTML video element with id="video"
const video = document.getElementById('video');
const texture = new THREE.VideoTexture(video);

anisotropy

anisotropy: number

The number of samples taken along the axis through the pixel that has the highest density of texels.

Remarks

A higher value gives a less blurry result than a basic mipmap, at the cost of more Texture samples being used.

Default

value of THREE.Texture.DEFAULT_ANISOTROPY. That is normally 1.
